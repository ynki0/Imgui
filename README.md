Imgui - это библиотека, которая позволяет интегрировать ImGui, популярный графический пользовательский интерфейс в режиме реального времени, с приложениями WinForms, использующими .NET Framework. 
Он обеспечивает отображение поверх DirectX 9 (DX9), что позволяет разработчикам с легкостью создавать интерактивные графические интерфейсы для своих приложений.

Простая интеграция: Легко интегрируйте ImGui с приложениями WinForms.
Интерактивные оверлеи: Поддерживает интерактивные оверлеи, которые можно переключать между интерактивным и неинтерактивным состояниями.
